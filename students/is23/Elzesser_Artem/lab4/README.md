# Отчет по ЛР 4: Веб-приложение с VK API

## Цель: Создание SPA для работы с данными VK через API с сортировкой и фильтрацией.

 ### Технологии

  *  VK API (methods: groups.getMembers, users.get)

  * JavaScript ES6+ (классы, модули)

   * Bootstrap 5 (UI компоненты)

  *  XMLHttpRequest (асинхронные запросы)

 ### Основной функционал

  -  Работа с VK API

   -   Получение списка участников группы

   -   Детальная информация о пользователе

   -   Обработка токенов и CORS

  ###  Сортировка участников (вариант 1)

 -  ID по возрастанию/убыванию

 -  Дата вступления (старые/новые)

 -  Динамическая пересортировка

  ###  Детальная страница пользователя

- Фото профиля (400px)

- Основная информация

- Статус онлайн/оффлайн

- Ссылка на VK профиль

 ### Интерфейс

Главная страница

   * Карточки участников (фото + имя)

  *  Выпадающий список сортировки

  *  Клик по карточке → детальная страница

Страница пользователя

  *  Крупное фото профиля

   * Город, дата рождения, статус

  *  Кнопка "Назад"

  *  Ссылка на оригинальный профиль VK

 Ключевые особенности

   * CORS обход через браузерное расширение

 *   Fallback данные при ошибках API

   * Динамический рендеринг карточек

   * Обработка ошибок VK API

  *  Адаптивный дизайн (Bootstrap Grid)
