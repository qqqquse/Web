# Отчет по ЛР 3: Веб-приложение на JavaScript

## Цель: Создание многостраничного SPA-приложения с модульной архитектурой и навигацией.

* Технологии

   - JavaScript ES6+ (модули, классы)

   - Bootstrap 5 (стилизация)

   - History API (навигация)

  -  npm (управление зависимостями)

* Основной функционал

  -  SPA-архитектура – навигация без перезагрузки страниц

  -  Роутер – обработка URL и управление страницами

  -  Главная страница – 4 карточки пород собак с адаптивным дизайном

  -  Детальная страница – полная информация о породе

  -  Компоненты – переиспользуемые UI-элементы

* Страницы
Главная

  -  Сетка карточек (4 породы)

   - Аккордеон с общей информацией

  -  Клик по карточке → детальная страница

* Детальная

  -  Фото породы

  -  Характеристики (рост, вес, темперамент)

   - Особенности ухода

  -  Кнопка возврата

 * Ключевые особенности

  -  Модульная организация кода (ES6 modules)

  -  История браузера через pushState/popstate

  -  Адаптивный дизайн (Bootstrap Grid)

  -  Компонентный подход

  -  Статические данные (4 породы собак)
