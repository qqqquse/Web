# Отчет по лабораторным работам "Калькулятор"
## Лабораторная работа 1: HTML/CSS

Цель работы: Знакомство с основами веб-разработки, изучение технологий HTML и CSS, создание интерактивного калькулятора с применением изученных концепций.
Выполненные задания:

   * Создана HTML-разметка калькулятора:

      -  Реализована структура с полем вывода и кнопками операций

      -  Добавлены элементы управления (селектор темы, информационные блоки)

  *  Разработан стиль с уникальным дизайном "CyberCalc":

      -  Применена тематика киберпанк с сине-фиолетовой палитрой

      -  Использован шрифт Orbitron для цифрового вида

     -   Реализована темная и светлая темы

  *  Дополнительные элементы:

      -  Добавлена кнопка с ссылкой на GitHub

      -  Реализованы сворачивающиеся блоки с информацией об авторе и группе

      -  Добавлено поле с описанием цели ЛР с подсветкой ключевых слов

# Лабораторная работа 2: JavaScript

## Цель работы: Реализация интерактивной логики калькулятора с использованием JavaScript.
Реализованный функционал:

   * Базовые операции:

     -   Сложение, вычитание, умножение, деление

       - Обработка деления на ноль

      -  Поддержка десятичных чисел

   * Расширенные математические операции:

       - Квадратный корень (√)

      -  Возведение в квадрат (x²)

      -  Факториал (x!) с ограничением до 20

      -  Проценты (%)

      -  Смена знака (±)

      -  Возведение в произвольную степень (Aᴮ)

  *  Дополнительные функции:

       - Накопительные операции для сложения и вычитания

      -  Кнопка добавления трех нулей (000)

      -  Управление цветом фона и дисплея

      -  Обработка backspace (⌫)

      -  Отображение ошибок (деление на 0, слишком большие числа)

   * Особенности реализации:

      -  Использование классов ES6 для организации кода

      -  Хранение состояния калькулятора в объекте

      - Обработка больших чисел с научной нотацией

      -  Адаптивный дизайн с плавными переходами

Технические особенности:

   * Архитектура: Класс Calculator с методами для каждой операции

   * Обработка событий: Event listeners для всех кнопок

   * Стилизация: CSS Grid для расположения кнопок

   * Обработка ошибок: Временное отображение сообщений об ошибках

   * Темы: Динамическое переключение между темной и светлой темами
